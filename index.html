每周推荐

IntelliJ IDEA Community Edition add

IntelliJ IDEA 是最为流行的 JAVA 开发工具之一，其智能编码辅助和人体工程学设计，让编写代码变得行云流水，令人愉快。

Eclipse IDE add

Eclipse IDE 是由 Eclipse 基金会支持的跨平台开源集成开发环境，基于其丰富的插件，支持JAVA、C/C++、Python、PHP、JavaScript等众多语言。

add

Visual Studio Code（简称VS Code）是一款由微软开发且跨平台的免费源代码编辑器。结合插件，VS Code 几乎能支持任何开发语言。免费轻量、插件丰富的特点最受开发者欢迎！

Facebook Time Card add

精准的时间对于服务器集群来说至关重要，Facebook 公司带来了一套新的开源时间设备：Time Card。 只有一张卡片大小、插在服务器PCIe插槽上，即可获得每天误差不超过一微秒的精准时间。

IT 自动化在当今企业中扮演着重要角色，红帽出品的 Ansible 带来了完美的解决方案。只需在一台机器上（甚至可以是笔记本电脑）安装 Ansible，便可通过 SSH 协议管理远程机器。
